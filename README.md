### ¿Qué es DataLake?
DataLeake es una aplicación web cliente orientada a la indexación, categorización y distribución controlada de recursos digitales técnicos. El sistema permite explorar un catálogo organizado por áreas de ingeniería y acceder a los archivos mediante enlaces temporales.
